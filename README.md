## 🎯 Purpose

This repository provides the **Appendix / Exercise Material (PDF)** for the book chapter:

**“Integrating Parallel and Distributed Computing (PDC) Concepts into Core Data Structures and Algorithms.”**

It is intended for instructors and students learning **OpenMP-based parallelism alongside traditional DSA topics.**

---

## 📄 PDF Contents

[The full appendix/exercise material is available here:](https://github.com/Reenie/PDC-in-Data-Structures-at-Utica-University/blob/main/Appendix_Exercise.md)

- Classroom exercise sheets  
- OpenMP sample codes (Hello World, Parallel Loops, Parallel Sum, etc.)  
- Assignments (Sort, Arrays, graph, etc.)

- Project guidelines and rubrics

## ✅ How to Use

1. Click the PDF link above or download `Appendix_Materials.pdf`.  
2. Open sample codes or assignments in **OnlineGDB** or a local compiler.  
3. Compile in OpenMP with fopenmp flag as follow:

   g++ filename.cpp -fopenmp
4. Modify, execute, and analyze performance (sequential vs parallel).
